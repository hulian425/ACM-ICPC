## Binary_Numbers_AND_Sum

**题意**

给你两个二进制序列，s1、s2 , 每次s2>>1; 结果 + s1&s2， 求最终结果

**题解**

这题很容易想到求每位的权值， 但怎么求呢？
因为s1只有为1的位才会对结果有影响，乘，s2 每位的前面1的个数（前缀和）

```c++
    rep(i, 1, m){
        sum[i] = sum[i-1];
        if (s2[i] == '1') sum[i] += 1;
    }
    int ans = 0;
    rep(i, 1, n)
    {
        if (s1[n-i+1] == '1')
        {
            if (m - i + 1 >= 1)
            ans = (ans + qmi(2, i - 1, 998244353)*(sum[m-i+1])%mod)%mod;
        }
    }
    cout << ans << endl;
```
