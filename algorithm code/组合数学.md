## Ayoub's_function

[Ayoub's_function](https://codeforces.com/contest/1301/problem/C)

**题意**

给出一个字符串中1的个数，和它的长度，其余都是0，然后构造一个字符串，使得f(s)最大，f(s)表示有多少个点对，
使得这个点对的区间中至少包含一个1。

**题解**

字符串的长度是n， 那么答案是`n*(n+1)/2` - 没有1的子串数（连续‘0’的子串）

```c++
        int n, m;
        cin >> n >> m;
        int k = (n-m)/(m+1);

        cout << (n+1)*n/2 - (k+1)*k/2 *(m+1) - (k+1)*((n-m)%(m+1))<< endl;
```
